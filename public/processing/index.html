These are sketches that I made using Processing software. Processing is an open-source graphics library which is based on Java. Processing is a fun and easy tool that can help one understand more about coding. It is a tool that can be easily explored and accessed by non-programmers. When using Processing, it seemed like it has limitations; however, once the concept has been created, almost anything was possible to visualize. Initially, I wanted to upload my works on to the web using Processing.js script. However, some of the works had interactive elements that the web could not support. So, my works are uploaded as in clips, and for those who want to take a look at how they are coded, please feel free to reach out.
